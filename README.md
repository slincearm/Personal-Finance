# 個人財務規劃 WebApp

這是一個功能完整的個人財務規劃 WebApp，專為手機 iOS WebApp 與現代桌面瀏覽器設計，協助您輕鬆管理每月可支配所得並進行投資、旅遊與生活費用的分配。

## 🌟 核心功能

*   **動態計算引擎**: 根據每月收入自動扣除固定支出與保險，精準計算出「可支配金額」，並依比例自動分配至「投資 (55%)」、「生活費 (35%)」與「旅遊 (10%)」。
*   **防呆鎖定 (Global Lock)**: 全域鎖定功能，一鍵將除收入外的所有欄位設為唯讀且禁止互動 (Hover/Focus失效)，防止誤觸修改。
*   **投資比例連動與信貸扣除**: 投資總額可先扣除「信用貸款」，剩餘的「可買股金額」再依據連動的台股與美股比例進行精準配置。
*   **多語系支援 (i18n)**: 內建正體中文（預設）與英文，並提供一鍵切換功能。
*   **本地資料儲存 (LocalStorage)**: 所有的收入、支出及設定皆會自動儲存於瀏覽器本地，下次開啟 App 時進度不流失。
*   **拖曳排序 (Drag and Drop)**: 支援個股卡片的上下拖曳排序，長按即可在清單中自由調整持股順序。
*   **可收折區塊 (Collapsible Sections)**: 投資、固定支出與保險支出等主要區塊皆提供點擊標題收折功能，並能獨立記憶展開狀態。
*   **現代化響應式介面**: 使用 Tailwind CSS 打造的精美卡片式 UI，完美支援深色模式 (Dark Mode)，並專門針對 iOS WebApp 提供 `100svh` 的滿版防縮放體驗。
## 🛠️ 技術棧

*   **框架**: React 18 + Vite
*   **語言**: TypeScript
*   **樣式**: Tailwind CSS v3
*   **圖示**: Lucide React
*   **多語系**: react-i18next
*   **拖曳套件**: dnd-kit

## 🚀 本地開發與執行

請確保您已安裝 Node.js (建議版本 18 以上)。

1.  **安裝依賴套件**
    ```bash
    npm install
    ```

2.  **啟動本地開發伺服器**
    ```bash
    npm run dev
    ```

3.  **編譯 (Production Build)**
    ```bash
    npm run build
    ```
